# Course Builder Quick Start Guide

Get up and running with the Admin Course Builder in 5 minutes.

## Step 1: Access the Builder (30 seconds)

1. Log in to your admin account
2. Navigate to **Courses** in the admin dashboard
3. Select the course you want to edit
4. Click **Edit Course** button

The Course Builder will open with your course ready to edit.

## Step 3: Add Your First Block (1 minute)

1. Select a lesson from the Course Structure (left panel)
2. In the Block Library (right panel), click on **Text Block**
3. Type your content in the editor
4. Use the toolbar to format (headings, bold, lists, etc.)
5. Click **Save**

Your block appears in the canvas and auto-saves!

## Step 4: Add More Content (2 minutes)

### Add a Video

1. Click **Video Block** in the Block Library
2. Choose **Embed** or **Upload**
3. For embed: Paste YouTube/Vimeo URL
4. For upload: Select video file (max 100MB)
5. Add a title for accessibility
6. Click **Save**

### Add an Image

1. Click **Image Block** in the Block Library
2. Drag and drop an image or click to browse (max 5MB)
3. Add **Alt Text** (required) - describe the image
4. Add optional caption
5. Click **Save**

### Add an Interactive Element

1. Switch to the **Interactive** tab in Block Library
2. Click **Poll Block**
3. Enter your question
4. Add 2-6 answer options
5. Click **Save**

## Step 5: Organize Your Content (30 seconds)

### Reorder Blocks
- Hover over a block
- Click and drag the handle on the left
- Drop in the new position

### Duplicate a Block
- Hover over a block
- Click the duplicate icon (two squares)
- Or press **Cmd+D** (Mac) / **Ctrl+D** (Windows)

### Delete a Block
- Hover over a block
- Click the delete icon (trash can)
- Confirm deletion

## Step 6: Preview Your Lesson (30 seconds)

1. Click **Preview** button in the top-right
2. See exactly how students will see your lesson
3. Test interactive elements
4. Click **Exit Preview** to return to editing

## Quick Reference

### Keyboard Shortcuts
| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Undo | Cmd+Z | Ctrl+Z |
| Duplicate | Cmd+D | Ctrl+D |
| Toggle AI Assistant | Cmd+G | Ctrl+G |

### File Limits
- Videos: 100MB max
- Images: 5MB max
- Supported video formats: MP4, WebM, MOV
- Supported image formats: JPG, PNG, GIF, WebP

### Block Categories

**Basic Blocks:**
- Text - Formatted content
- Video - Embed or upload
- Image - With caption and alt text
- Code - Syntax highlighted
- List - Bullet, numbered, or checkbox
- Divider - Visual separator

**Interactive Blocks:**
- Reflection - Written responses
- Poll - Multiple choice voting
- Word Cloud - Collective visualization
- AI Generator - AI-assisted content
- And 12 more specialized tools!

## Common Issues

### Changes Not Saving
- Check internet connection
- Look for error message in top-right
- Try refreshing the page

### Upload Failed
- Check file size (100MB videos, 5MB images)
- Verify file format is supported
- Try a different file

### Can't Drag Blocks
- Make sure you're dragging from the handle
- Try refreshing the page
- Check if block is locked
