# Course Builder Video Tutorial Script

## Video 1: Getting Started (5 minutes)

### Introduction (30 seconds)
**[Screen: Admin dashboard]**

"Welcome to the Admin Course Builder tutorial. In this video, you'll learn how to access the course builder and understand its interface. Let's get started."

### Accessing the Builder (1 minute)
**[Screen: Navigate from admin dashboard to course list]**

"First, log in to your admin account and navigate to the Courses section. Here you'll see all your courses. Click on any course you want to edit, then click the 'Edit Course' button to open the Course Builder."

**[Screen: Course builder loads]**

"The builder will load with your course structure ready to edit. Notice the three-panel layout we'll explore next."

### Interface Overview (2 minutes)
**[Screen: Highlight each panel]**

"The Course Builder has three main panels:

On the left, the Course Structure panel shows your modules and lessons in an expandable accordion format. The currently selected lesson is highlighted in blue.

In the center, the Canvas is where you'll arrange and edit your content blocks. This is your main workspace.

On the right, the Block Library contains all available block types you can add to your lessons. It's organized into Basic and Interactive tabs."

**[Screen: Show mobile view]**

"On smaller screens, the side panels collapse into mobile menus accessible from the header buttons."

### Auto-Save Feature (1 minute)
**[Screen: Make a change and show save indicator]**

"One of the most important features is auto-save. The builder automatically saves your changes every 2 seconds after you stop editing. Watch the indicator in the top-right corner:

- 'Saving...' means your changes are being saved
- 'Saved' with a checkmark means everything is saved
- If you see an error, your changes are preserved locally and you can retry

Always wait for the 'Saved' indicator before closing the browser tab."

### Conclusion (30 seconds)
**[Screen: Full interface view]**

"Now you understand the basic interface and auto-save feature. In the next video, we'll learn how to add and configure content blocks. Thanks for watching!"

## Video 2: Adding and Configuring Blocks (8 minutes)

### Introduction (30 seconds)
**[Screen: Course builder with empty lesson]**

"In this video, you'll learn how to add blocks to your lessons and configure them. We'll cover both basic content blocks and interactive elements."

### Adding a Text Block (2 minutes)
**[Screen: Block library, Basic tab]**

"Let's start by adding a text block. In the Block Library on the right, make sure you're on the Basic tab. Click on 'Text Block' to add it to your lesson."

**[Screen: Text block modal opens]**

"The configuration modal opens automatically. Here you can use the rich text editor to format your content. Let's add a heading and some paragraphs."

**[Screen: Type content, use formatting toolbar]**

"Use the toolbar to add headings, make text bold or italic, create lists, and add links. When you're done, click Save."

**[Screen: Block appears in canvas]**

"The block now appears in your lesson canvas. It will auto-save in a few seconds."

### Adding a Video Block (2 minutes)
**[Screen: Add video block from library]**

"Next, let's add a video. Click 'Video Block' in the library."

**[Screen: Video block modal]**

"You have two options: embed a video from YouTube or Vimeo, or upload your own video file. Let's try embedding a YouTube video."

**[Screen: Select embed, paste URL]**

"Select 'Embed', paste your YouTube URL, and add a title for accessibility. Click Save."

**[Screen: Video block in canvas]**

"The video block appears with a preview. Students will be able to watch it directly in the lesson."

### Adding an Image Block (1.5 minutes)
**[Screen: Add image block]**

"Now let's add an image. Click 'Image Block' in the library."

**[Screen: Image block modal]**

"You can drag and drop an image file or click to browse. Remember, images must be under 5MB."

**[Screen: Upload image, add alt text and caption]**

"After uploading, add alt text - this is required for accessibility. Describe what's in the image. You can also add an optional caption. Click Save."

### Adding an Interactive Block (1.5 minutes)
**[Screen: Switch to Interactive tab]**

"Let's add an interactive element. Switch to the Interactive tab in the Block Library. Let's add a Poll."

**[Screen: Poll block modal]**

"Enter your poll question and add at least two options. You can add up to six options. Click Save."

**[Screen: Poll block in canvas]**

"The poll block appears in your canvas. Students will be able to vote and see results."

### Conclusion (30 seconds)
**[Screen: Lesson with multiple blocks]**

"Great! You've learned how to add text, video, image, and interactive blocks. In the next video, we'll learn how to organize and manage your blocks. See you there!"

## Video 5: Advanced Tips and Best Practices (7 minutes)

### Introduction (30 seconds)
**[Screen: Course builder interface]**

"In this video, we'll cover advanced tips and best practices to help you create effective, engaging courses. Let's dive in!"

### Content Organization (1.5 minutes)
**[Screen: Well-structured course]**

"Start by planning your course structure before adding blocks. Organize content into logical modules and lessons."

**[Screen: Example lesson with mixed blocks]**

"Within lessons, mix different block types. Combine text explanations with videos, images, and interactive elements. This supports different learning styles."

**[Screen: Show divider blocks]**

"Use divider blocks to create visual breaks between major sections. This improves readability and helps students process information."

### Engagement Strategies (1.5 minutes)
**[Screen: Lesson with interactive blocks]**

"Add at least one interactive block per lesson to keep students engaged. Polls and reflection blocks work great for this."

**[Screen: Show reflection block after video]**

"Place reflection blocks after key concepts to encourage critical thinking. For example, after a video, ask students to reflect on what they learned."

**[Screen: Show poll for formative assessment]**

"Use polls for quick formative assessment. This helps you gauge understanding without the pressure of a graded quiz."

### Accessibility Best Practices (1.5 minutes)
**[Screen: Image block modal with alt text]**

"Always provide descriptive alt text for images. Be specific about what's in the image. For example, 'Bar chart showing sales growth from 2020-2025' is better than just 'chart'."

**[Screen: Video block with title]**

"Add titles to all videos for accessibility. If possible, ensure videos have captions."

**[Screen: Text block with heading structure]**

"In text blocks, use proper heading hierarchy. Start with H2 for main sections, H3 for subsections, and so on. This helps screen reader users navigate content."

### Performance Optimization (1.5 minutes)
**[Screen: Image optimization example]**

"Optimize images before uploading. Compress them to reduce file size while maintaining quality. This improves loading times for students."

**[Screen: Video embed vs upload]**

"For videos, consider using embed instead of upload when possible. This saves storage space and often provides better streaming performance."

**[Screen: Lesson with appropriate block count]**

"Aim for 10-20 blocks per lesson. Too many blocks can overwhelm students, while too few might not provide enough content."

### Keyboard Shortcuts (1 minute)
**[Screen: Show keyboard shortcuts in action]**

"Master these keyboard shortcuts for efficiency:
- Cmd+Z or Ctrl+Z to undo
- Cmd+D or Ctrl+D to duplicate a block
- Tab to navigate between elements
- Enter to activate buttons"

### Troubleshooting Common Issues (1 minute)
**[Screen: Show error states and solutions]**

"If uploads fail, check your file size and format. Videos must be under 100MB, images under 5MB."

**[Screen: Show save indicator]**

"If auto-save isn't working, check your internet connection. The error indicator will show if there's a problem."

**[Screen: Show browser refresh]**

"If drag-and-drop stops working, try refreshing the page. Your changes are saved, so you won't lose work."

### Conclusion (30 seconds)
**[Screen: Completed course]**

"These tips will help you create professional, accessible, and engaging courses. Remember to check the user guide for more detailed information. Happy teaching!"

## Production Notes

### Equipment Needed
- Screen recording software (e.g., Camtasia, ScreenFlow, OBS)
- Microphone for clear audio
- Video editing software
- Sample course with content for demonstrations

### Recording Tips
1. Record in 1920x1080 resolution
2. Use a consistent cursor highlight
3. Speak clearly and at a moderate pace
4. Pause between sections for easier editing
5. Record in a quiet environment
6. Use a pop filter to reduce plosives

### Editing Guidelines
1. Add intro/outro graphics with branding
2. Include chapter markers for easy navigation
3. Add captions for accessibility
4. Export in multiple formats (MP4, WebM)
5. Optimize for web streaming
6. Include downloadable transcripts

### Publishing Checklist
- [ ] All videos recorded and edited
- [ ] Captions added to all videos
- [ ] Transcripts created
- [ ] Videos uploaded to hosting platform
- [ ] Thumbnails created for each video
- [ ] Video links added to user guide
- [ ] Playlist created for sequential viewing
- [ ] Feedback mechanism implemented

### Video Hosting Recommendations
- YouTube (public or unlisted)
- Vimeo (for private hosting)
- Self-hosted with video player
- LMS integration if available
