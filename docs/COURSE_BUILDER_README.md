## üéì Block Types Quick Reference

### Basic Blocks (6 types)
1. **Text** - Rich formatted content
2. **Video** - Embed or upload videos
3. **Image** - Images with captions
4. **Code** - Syntax-highlighted code
5. **List** - Bullet, numbered, checkbox lists
6. **Divider** - Visual separators

### Interactive Blocks (16 types)
1. **Reflection** - Written reflections
2. **Poll** - Multiple choice voting
3. **Word Cloud** - Collective visualization
4. **AI Generator** - AI-assisted content
5. **Choice Comparison** - Compare options
6. **Design Fixer** - Fix design problems
7. **Player Type Simulator** - Gamification types
8. **Reward Schedule Designer** - Reward schedules
9. **Flow Channel Evaluator** - Flow state assessment
10. **Pitch Analysis Generator** - Pitch analysis
11. **Narrative Generator** - Story creation
12. **Dark Pattern Redesigner** - Ethical design
13. **ROE Dashboard** - Education analytics
14. **Journey Timeline** - Learning journeys
15. **Certificate Generator** - Completion certificates
16. **Final Assessment** - Comprehensive tests

See [Block Types Reference](BLOCK_TYPES_REFERENCE.md) for detailed information on each.

## üìè Technical Specifications

### File Limits
- **Videos:** 100MB maximum
- **Images:** 5MB maximum

### Supported Formats
- **Videos:** MP4, WebM, MOV
- **Images:** JPG, PNG, GIF, WebP
- **Video Embeds:** YouTube, Vimeo

### Browser Support
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Recommended Specifications
- **Screen Size:** 1024px width minimum (tablet/desktop)
- **Internet:** Stable connection for auto-save
- **RAM:** 4GB minimum, 8GB recommended

## üéâ Quick Tips

1. **Auto-save is your friend** - Wait for "Saved" before closing
2. **Preview often** - See how students will experience your content
3. **Mix block types** - Combine text, media, and interactive elements
4. **Use alt text** - Make content accessible to all learners
5. **Optimize files** - Compress images and videos before upload
6. **Start simple** - Master basic blocks before advanced features
7. **Keyboard shortcuts** - Speed up your workflow
8. **Plan first** - Outline your course structure before building
